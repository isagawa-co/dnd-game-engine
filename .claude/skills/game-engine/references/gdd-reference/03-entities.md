# 3. Entities — Tiny Civ Reference

## Civilizations
Each civilization has:
- **Name** — e.g., "The Riverfolk", "The Iron Horde"
- **Unique Ability** — passive bonus (e.g., +1 food from rivers)
- **Unique Unit** — replaces a standard unit with better stats
- **Unique Building** — replaces a standard building with bonus yield
- **Start Bias** — preferred terrain for spawn (coast, river, plains)

## Promotions
XP from combat. Unlock at 10, 30, 60 XP:

| Promotion | Effect |
|-----------|--------|
| Shock | +25% in open terrain |
| Drill | +25% in rough terrain |
| March | Heal every turn, even if moved |
| Blitz | Can attack twice per turn |
| Siege | +50% vs cities |
| Medic | Heal adjacent friendly units |
