# 7. Win/Lose Conditions — Tiny Civ Reference

## Victory Conditions (6 paths)

| Victory | Trigger |
|---------|---------|
| Domination | Control all original capitals |
| Science | Research all Modern era techs + build Space Program project |
| Culture | Adopt all policies in 3 complete policy trees |
| Religion | Your religion is dominant in all civilizations |
| Diplomatic | Win a vote in World Congress (unlocked when all civs met) |
| Score | Highest score when turn limit reached (fallback) |

## Score Calculation
Score = population + territory + techs + wonders + military strength
