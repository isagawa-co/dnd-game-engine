# 18. Scope/Phases — Tiny Civ Reference

## Development Phases

### Phase 1 — Core (Playable Prototype)
- Map generation and rendering
- Units with movement and basic combat
- Cities with founding/growth/production
- Fog of war
- Basic AI
- Turn system
- Input handling

### Phase 2 — Depth
- Tech tree and research
- Buildings and improvements
- Tile yields and resources
- Barbarians
- Happiness system

### Phase 3 — Systems
- Diplomacy and trade
- Policies and governments
- Religion
- Great people
- Wonders

### Phase 4 — Polish
- Victory conditions
- History/timeline
- Random events
- AI personality tuning
- Balance pass
- Multiple civ definitions
