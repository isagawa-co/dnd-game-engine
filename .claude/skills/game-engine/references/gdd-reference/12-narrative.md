# 12. Narrative/Lore — Tiny Civ Reference

## History Timeline
The game tracks major events and builds a history log:

| Event Type | Example |
|-----------|---------|
| City founded | "The Riverfolk founded Riverside in 3200 BC" |
| Wonder built | "The Iron Horde completed Stonehenge in 2800 BC" |
| War declared | "The Scholars declared war on the Merchants in 1200 AD" |
| Tech discovered | "The Prophets discovered Gunpowder in 1450 AD" |
| Religion founded | "The Way of the Sun was founded in Sunhaven" |
| Great person born | "A Great Scientist was born in Ironhold" |
| City captured | "Riverside was conquered by the Iron Horde" |

## Era Progression
- Game era advances when any civ researches a tech in the next era
- Global announcement
- Visual theme shift in ASCII rendering (tile character styling changes per era)
