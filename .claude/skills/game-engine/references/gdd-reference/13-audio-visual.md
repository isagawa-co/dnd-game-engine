# 13. Audio/Visual Style — Tiny Civ Reference

## Art Direction
- Hand-drawn ASCII characters
- Dwarf Fortress art style
- Minimalist — distill Civilization into tight, fast-playing package
- Full game takes 30–60 minutes

## Rendering Layers
1. **Terrain layer** — land, water, natural features (`.` `^` `▲` `♣` `~` `≈` `░`)
2. **Entity layer** — units, cities, improvements (rendered on top of terrain)

## Era-Themed Styling
- Tile character styling changes subtly per era
- Visual progression reflects technological advancement
- ASCII characters remain readable across all eras

## Fog States
- **Unexplored** — solid black blocks
- **Revealed** — dimmed/muted colors
- **Visible** — full brightness
