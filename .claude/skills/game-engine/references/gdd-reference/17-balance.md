# 17. Balance Levers — Tiny Civ Reference

## Tuning Knobs

| Lever | Purpose | Example |
|-------|---------|---------|
| Yield multipliers per game speed | Keep pace feeling right | Quick = ×2 yields |
| Unit cost vs strength curve | Prevent snowballing | Swordsman 60/18 vs Tank 150/50 |
| Happiness thresholds | Limit unchecked expansion | -3 happiness per city |
| Tech cost scaling | Gate era progression | Ancient ~20, Modern ~800 |
| AI personality weights | Create distinct opponents | Warlord: aggression 9 |
| Wonder costs | Risk vs reward | 100 (Ancient) to 300 (Modern) |
| Trade route yields | Incentivize diplomacy over war | Gold based on distance + city size |
| Barbarian spawn rate | Early game pressure | Scales with era |
| Great person thresholds | Reward specialization | Points from specialist buildings |
