# 16. Configuration — Tiny Civ Reference

## Game Configuration Options

| Setting | Values | Default |
|---------|--------|---------|
| Map size | Tiny (30×20), Small (50×35), Standard (80×50) | Small |
| Number of civs | 2 to 8 | 4 |
| Difficulty | Settler, Chieftain, Prince, King, Emperor | Prince |
| Game speed | Quick (×2), Standard, Epic (×0.5) | Standard |
| Victory types | Toggle each on/off | All on |
| Barbarians | On/Off | On |

## Difficulty Scaling
Difficulty affects AI bonuses (not AI intelligence):
- Extra starting units
- Yield percentage bonuses
- Combat strength bonus

Higher difficulty = stronger AI with more resources, not smarter decisions.
