# 8. UI/Rendering — Tiny Civ Reference

## Main View
ASCII map with hand-drawn characters. Cities as stars, warriors as letters, fog as solid blocks. Status bar at bottom: turn count, gold income, science output, current city production.

## Panels

| Panel | Content |
|-------|---------|
| Map | Main game view, scrollable |
| City | Population, buildings, production queue, tile yields |
| Tech | Tech tree, current research, turns remaining |
| Civics | Policy trees, adopted policies, culture progress |
| Diplomacy | Known civs, relations, trade offers |
| Military | Unit list, promotions, combat previews |
| History | Timeline of events |
| Score | Victory progress for each condition |

## Input (Keyboard-Driven)

| Key | Action |
|-----|--------|
| Arrow / WASD | Pan map |
| Enter | Select / confirm |
| Esc | Back / cancel |
| Number keys | Quick actions |
| Tab | Cycle units needing orders |
| Space | End turn |
